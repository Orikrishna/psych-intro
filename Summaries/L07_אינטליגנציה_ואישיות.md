## מהם הבדלים בין-אישיים?

[📺 צפי בסרטון](https://www.youtube.com/watch?v=25LOIF-oT-s)

### הגדרה
**הבדלים בין-אישיים (Individual Differences)** = תחום מחקר שעוסק בתכונות ויכולות שבהן אנשים שונים זה מזה.

### מה נחקר?
- יכולות קוגניטיביות (אינטליגנציה)
- תכונות אישיות
- יכולות אתלטיות, מוזיקליות
- חברותיות, סקרנות, נטילת סיכונים
- זיכרון עבודה, זיהוי פנים

### איך מודדים?
באמצעות **כלי מדידה** שנותנים ערך מספרי לתכונה הנמדדת.

**דוגמה - שאלון נטילת סיכונים:**
- "אני אוהב/ת ריגושים"
- "אני מחפש/ת הרפתקאות"
- "אני רוצה לנסות כל דבר פעם בחיים"

### שני מדדים חשובים לכלי מדידה

| מדד | הגדרה | דוגמה |
|-----|-------|-------|
| **מהימנות (Reliability)** | האם כלי המדידה מדויק ונותן תוצאות עקביות? | משקל שנותן אותו מספר בכל פעם |
| **תוקף (Validity)** | האם כלי המדידה מודד את מה שהתכוון למדוד? | מבחן IQ שמודד אינטליגנציה ולא לחץ |

**חשוב:** כלי מדידה יכול להיות מהימן אבל לא תקף!

## אינטליגנציה - הגדרה ומדידה

[📺 צפי בסרטון](https://www.youtube.com/watch?v=25LOIF-oT-s&t=148)

### הניסיון הראשון - פרנסיס גלטון (סוף המאה ה-19)

**הגדרת אינטליגנציה לפי גלטון:** כישורים סנסוריים ומוטוריים.

**מה מדד?**
- זמן תגובה
- ראייה, שמיעה, ראיית צבע
- חוזק נשיפה
- יכולת למשוך/למחוץ

**התוצאה:**
- ✅ המבחנים היו **מהימנים** - נתנו תוצאות עקביות
- ❌ המבחנים לא היו **תקפים** - אנשים "אינטליגנטים" לא קיבלו ציונים גבוהים יותר

**תרומותיו החשובות של גלטון:**
- פיתח את מדד ה**מתאם** (קשר בין משתנים)
- היה הראשון שהשתמש ב**מחקרי תאומים**
- בן דודו של צ'ארלס דרווין!

## מבחני אינטליגנציה

[📺 צפי בסרטון](https://www.youtube.com/watch?v=TAUnkEiZ98Q)

### אלפרד בינה (1905) - המבחן הראשון המוצלח

**הרקע:** משרד החינוך הצרפתי ביקש לאתר ילדים עם קשיים לימודיים.

**הגדרת אינטליגנציה לפי בינה:** יכולת חשיבה ופתרון בעיות.

**מה כלל המבחן?**
- יכולות מילוליות
- יכולות כמותיות
- חשיבה ויזואלית
- זיכרון לטווח קצר

**דוגמאות לשאלות לפי גיל:**
| גיל | שאלות |
|-----|-------|
| 3 | הצבע על אף, פה ועיניים; חזור על שתי ספרות |
| 6 | הגדר מילים לפי שימוש; ספור 13 מטבעות |

### מנת המשכל - IQ

**נוסחה מקורית:**
```
IQ = (גיל מנטלי / גיל כרונולוגי) × 100
```

**דוגמה:** ילד בן 4 עם יכולות של בן 3 → IQ = 75

### לואיס טרמן ומבחן סטנפורד-בינה (שנות ה-50)

**שיפורים:**
- התאמה גם לילדים מוכשרים (לא רק לאיתור בעיות)
- שינוי בשיטת הציון - ציון יחסי לקבוצת הגיל

### דייויד וקסלר - WAIS & WISC

**החידוש:** הוספת חלק **ביצועי** בנוסף לחלק המילולי.

```
┌─────────────────────────────────────────────────────────────────┐
│                    מבחן וקסלר לאינטליגנציה                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   חלק מילולי:                    חלק ביצועי:                   │
│   • הבנה מילולית                 • ארגון מרחבי                  │
│   • תרגילי חשבון                 • בניית דגם מקוביות            │
│   • זיכרון עבודה                 • פאזל                        │
│   • אוצר מילים                   • השוואת צורות                 │
│                                                                 │
│                    ↓                                            │
│              ציון IQ כללי + 2 ציונים נפרדים                     │
└─────────────────────────────────────────────────────────────────┘
```

| מבחן | קיצור | לגילאים |
|------|-------|---------|
| Wechsler Adult Intelligence Scale | WAIS | מבוגרים |
| Wechsler Intelligence Scale for Children | WISC | ילדים |

### התפלגות ציוני IQ

```
                    עקומת הפעמון
                        ╱╲
                       ╱  ╲
                      ╱    ╲
                     ╱      ╲
                    ╱        ╲
                   ╱          ╲
──────────────────────────────────────────
   70        85      100      115      130
   ↑                  ↑                  ↑
 פיגור             ממוצע             מחוננים
(2.5%)             (50%)             (2.5%)
```

### תיאוריית G של ספירמן

**הטענה:** אינטליגנציה = יכולת כללית (G) + יכולות ספציפיות (S)

**משל הספורטאים:**
- לכל ספורטאי יש **יכולת ספציפית** (טניס, שחייה, כדורסל)
- אבל לכל הספורטאים יש **יכולת אתלטית כללית** גבוהה יותר מלא-ספורטאים

**אותו דבר באינטליגנציה:**
- אפשר להיות חזק יותר במילולי או בביצועי
- אבל בדרך כלל מי שחזק באחד חזק גם בשני

## תורשה וסביבה באינטליגנציה

[📺 צפי בסרטון](https://www.youtube.com/watch?v=vEJuaqsU_3I)

### גנטיקה התנהגותית

**השאלה:** ממה נובעים ההבדלים באינטליגנציה - תורשה או סביבה?

**השיטה:** מחקרי תאומים ומשפחות.

### השוואת מתאמים

```
מתאם = מדד לקשר בין משתנים (0 = אין קשר, 1 = קשר מושלם)
```

**גרף פיזור (Scatter Plot):**
```
מתאם גבוה:           מתאם נמוך:
    IQ תאום ב'           IQ תאום ב'
         •  •                 •
        •  •              •     •
       •  •            •    •
      •  •                •  •
     •  •             •      •
   ───────────         ───────────
     IQ תאום א'          IQ תאום א'
```

### ממצאים ממחקרי תאומים

| קשר משפחתי | גנים משותפים | סביבה משותפת | מתאם IQ |
|------------|-------------|--------------|---------|
| תאומים זהים שגדלו יחד | 100% | 100% | 0.85 |
| **תאומים זהים שהופרדו בלידה** | 100% | 0% | **0.76** |
| תאומים לא זהים שגדלו יחד | 50% | 100% | 0.60 |
| אחים מאומצים | 0% | 100% | 0.25 |
| ילד מאומץ והורה מאמץ | 0% | 100% | ~0 |

### מה אפשר ללמוד?

```
┌─────────────────────────────────────────────────────────────────┐
│              ממצאים מרכזיים על תורשה וסביבה                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  1. תאומים זהים שהופרדו בלידה עדיין דומים מאוד!               │
│     → לתורשה יש תרומה מכרעת                                    │
│                                                                 │
│  2. המתאם בין תאומים זהים שהופרדו (0.76) >                     │
│     תאומים לא זהים שגדלו יחד (0.60)                            │
│     → 100% גנים > 50% גנים + 100% סביבה                        │
│                                                                 │
│  3. אין מתאם בין הורים מאמצים לילדים מאומצים בבגרות            │
│     → לסביבה המשותפת יש תפקיד קטן                              │
│                                                                 │
│  4. בבגרות התרומה של הסביבה יורדת                              │
│     → בילדות בית הספר משפיע על ציון IQ                         │
└─────────────────────────────────────────────────────────────────┘
```

### אינטראקציה בין תורשה לסביבה

**חשוב להבין:** תורשה וסביבה לא בלתי תלויים!
- אם יש לי נטייה גנטית לספורט → אעשה יותר ספורט → אשתפר
- אם יש לי כישרון גנטי לריצה אבל אף פעם לא רצתי → הכישרון לא יבוא לידי ביטוי

## אינטליגנציות מרובות ורגשית

[📺 צפי בסרטון](https://www.youtube.com/watch?v=oRNcdx6f5UQ)

### הווארד גארדנר - אינטליגנציות מרובות

**הטענה:** אינטליגנציה קוגניטיבית היא רק רכיב אחד!

**7 סוגי אינטליגנציה:**
1. מילולית-לשונית
2. לוגית-מתמטית
3. **קואורדינציה גופנית** (ספורטאים, רקדנים)
4. **מוזיקלית** (נגנים, מלחינים)
5. מרחבית
6. **בין-אישית** (הבנת אחרים)
7. **תוך-אישית** (הבנת עצמי)

**דוגמאות:** מוצארט, הביטלס, מראדונה, מייקל ג'ורדן - לא יודעים מה ה-IQ שלהם, אבל יש להם כישורים יוצאי דופן!

### אינטליגנציה רגשית - ג'ון מאייר

**הגדרה - 4 יכולות:**
1. **תפיסה והבעה** מדויקת של רגשות
2. **יצירת רגשות** שמשרתים את החשיבה
3. **הבנת רגשות** - מה גורם למה
4. **ניהול רגשות** - ויסות עצמי

### מבחן MSCEIT

**איך מודדים?**

| יכולת | שאלה לדוגמה |
|-------|------------|
| תפיסה | מה הרגש שהפנים בתמונה מביעות? |
| יצירה | בפגישה ראשונה עם הורים של בן/בת זוג - איזה רגש יועיל? |
| הבנה | תום היה לחוץ והבוס ביקש ממנו עוד מטלה - מה ירגיש? |
| ניהול | דבי חזרה מחופשה רגועה - מה תעשה כדי להמשיך להרגיש טוב? |

**איך קובעים תשובה נכונה?**
- מה רוב האנשים עונים
- מה מומחים לרגשות עונים

### מה קשור לאינטליגנציה רגשית?

| קשר חיובי | קשר שלילי |
|-----------|-----------|
| הצלחה בלימודים (מעבר ל-IQ) | שימוש בסמים |
| מערכות יחסים טובות | התנהגות אלימה |
| התמודדות עם לחץ | - |
| מקובלות חברתית | - |

**אזהרה של מאייר:** הספרות הפופולרית הגזימה בטענות על אינטליגנציה רגשית. צריך לקרוא מאמרים מדעיים!

## אישיות - תיאוריות ומדידה

[📺 צפי בסרטון](https://www.youtube.com/watch?v=hzLGkGIvMIQ)

### גישות היסטוריות

| תיאורטיקן | הטענה |
|-----------|-------|
| **פרויד** | אישיות מונעת מדחפים תוקפניים ומיניים לא מודעים |
| **קארל רוג'רס** | יש לאנשים נטייה מולדת לגדילה ולשינויים חיוביים |

**הבעיה:** התיאוריות מבוססות על ניסיון קליני, לא על מחקר אמפירי.

### הגדרה אמפירית

**אישיות** = אוסף של תכונות
**תכונה** = נטייה יציבה יחסית להתנהג בדרך מסוימת

### מדידת אישיות - גישות שנכשלו

**דוגמה היסטורית:** שיפוט לפי תווי פנים
- צ'ארלס דרווין כמעט לא עלה לספינת הביגל כי לקפטן לא מצא חן האף שלו!

## חמש התכונות הגדולות - Big Five

[📺 צפי בסרטון](https://www.youtube.com/watch?v=hzLGkGIvMIQ&t=175)

### איך התגלו?
באמצעות **Factor Analysis** - שיטה סטטיסטית שמוצאת קשרים בין אלפי תכונות.

**התוצאה:** לא משנה אם מתחילים עם 4,000 או 60 תכונות - תמיד נשארים עם 5!

### חמש התכונות

```
┌─────────────────────────────────────────────────────────────────┐
│                      The Big Five                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  O - Openness to Experience   │  פתיחות לחוויות               │
│  C - Conscientiousness        │  מצפוניות/מכוונות              │
│  E - Extraversion             │  מוחצנות                       │
│  A - Agreeableness            │  נעימות                        │
│  N - Neuroticism              │  נוירוטיות/יציבות רגשית        │
│                                                                 │
│                    (OCEAN)                                      │
└─────────────────────────────────────────────────────────────────┘
```

### פירוט התכונות

| תכונה | מאפיינים | פריטים לדוגמה |
|-------|----------|---------------|
| **מוחצנות (E)** | צורך חברתי, דברנות, פעלתנות | "אני דברן", "אני אנרגטי" |
| **נוירוטיות (N)** | נטייה לחרדה, דיכאון, עצבנות | "אני מרבה לדאוג", "אני מתוח" |
| **נעימות (A)** | אדיבות, אמון, מזג נוח | "אני סלחן", "אני מתחשב" |
| **מצפוניות (C)** | אחריות, יסודיות, משמעת עצמית | "אני יסודי", "אני מתכנן" |
| **פתיחות (O)** | דמיון, סקרנות, אופקים רחבים | "אני סקרן", "אני מקורי" |

### מה התכונות מנבאות?

| מנבא | תכונה קשורה |
|------|-------------|
| ציונים בבית הספר | מצפוניות (C) |
| ציון פסיכומטרי (מעבר לציונים) | פתיחות (O) |
| אינטליגנציה רגשית | נעימות (A), פתיחות (O), מצפוניות (C) |

### תורשה וסביבה באישיות

**ממצא מעניין:**
- יש מתאם בין תאומים זהים גם בתכונות אישיות
- **אבל** המתאם נמוך יותר מאשר באינטליגנציה

**למה?**
- תכונות אישיות מושפעות גם מ**סביבה לא משותפת**
- חברים שונים, חוגים שונים, התייחסות שונה

### מבחן המרשמלו - יציבות האישיות

[📺 צפי בסרטון](https://www.youtube.com/watch?v=hzLGkGIvMIQ&t=594)

**הניסוי (וולטר מישל):**
- ילדים בני 4 יושבים עם מרשמלו
- אם לא יאכלו עד שהנסיין חוזר - יקבלו עוד אחד
- מודדים **יכולת דחיית סיפוקים**

**מעקב אחרי 15 שנה - מי שדחה סיפוקים בגיל 4:**
- יכולת ריכוז גבוהה יותר
- שליטה עצמית גבוהה יותר
- ציונים גבוהים יותר בפסיכומטרי
- צריכה קטנה יותר של סמים

**מעקב אחרי 40 שנה:**
- סריקת fMRI הראתה הבדלים במוח!
- אזורים הקשורים לשליטה עצמית פועלים אחרת

**המסקנה:** אישיות יציבה לאורך עשרות שנים!

## חקר אישיות תינוקות

[📺 צפי בסרטון](https://www.youtube.com/watch?v=J74VneLdLlw)

### ביקור במרכז אדלר להתפתחות הילד

**איך מודדים אישיות אצל תינוקות?** (אי אפשר לתת שאלון!)
- מבחנים התנהגותיים

### משימות למדידת אישיות תינוקות

**משימה 1: צעצוע בקופסה סגורה**
- שמים צעצוע בקופסה שקשה לפתוח
- מודדים:
  - מידת המצוקה (סקאלה -3 עד +3)
  - התגובה הרגשית הכללית (0-4)

**משימה 2: התנהגויות ויסות**
מה התינוק עושה כשמתוסכל?

| התנהגות | דוגמה |
|---------|-------|
| התעסקות בחפץ | ניסיון להגיע, לפתוח |
| הסחת דעת עצמית | בוהה למקום אחר |
| פנייה להורה | מבקש עזרה |
| אגרסיה | דפיקה, רקיעה |

**משימה 3: פרידה מההורה**
- האם יוצאת מהחדר ל-2 דקות
- מודדים:
  - רמת חרדה (1-7)
  - יכולת התמודדות (1-7)
  - תגובה לחזרת ההורה (1-7)

### למה זה חשוב?
- מנבא התפתחות עתידית
- קשור להפרעות שינה
- עוזר להבין טמפרמנט מולד

## סיכום ומושגי מפתח

### טבלת מושגי מפתח

| מושג | הגדרה |
|------|-------|
| הבדלים בין-אישיים | תחום שחוקר תכונות ויכולות שבהן אנשים שונים |
| מהימנות (Reliability) | מידת הדיוק והעקביות של כלי מדידה |
| תוקף (Validity) | האם הכלי מודד את מה שהתכוון למדוד |
| IQ | מנת משכל - ציון יחסי לקבוצת הגיל |
| G (General) | יכולת אינטליגנציה כללית (ספירמן) |
| WAIS | מבחן וקסלר למבוגרים |
| WISC | מבחן וקסלר לילדים |
| גנטיקה התנהגותית | תחום שחוקר תרומת תורשה וסביבה להבדלים |
| מתאם | מדד לקשר בין משתנים (0-1) |
| אינטליגנציה רגשית | יכולת לתפוס, להבין ולנהל רגשות |
| Big Five | 5 תכונות האישיות המרכזיות |
| OCEAN | ראשי תיבות ל-Big Five |
| מבחן המרשמלו | מבחן ליכולת דחיית סיפוקים |

### 📝 נקודות חשובות למבחן

1. **מהימנות ≠ תוקף** - מבחן גלטון היה מהימן אבל לא תקף
2. **IQ = ציון יחסי** - מספר שמראה מיקום ביחס לקבוצת הגיל
3. **וקסלר הוסיף חלק ביצועי** - לא רק מילולי
4. **תורשה > סביבה באינטליגנציה** - תאומים זהים שהופרדו עדיין דומים
5. **אינטליגנציה רגשית** - 4 יכולות: תפיסה, יצירה, הבנה, ניהול
6. **Big Five** - OCEAN: פתיחות, מצפוניות, מוחצנות, נעימות, נוירוטיות
7. **אישיות יציבה** - מבחן המרשמלו מנבא התנהגות 40 שנה אחרי!
8. **באישיות הסביבה חשובה יותר** - גם סביבה לא משותפת משפיעה

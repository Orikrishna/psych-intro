## הקשר בין גוף לנפש
[📺 צפי בסרטון](https://www.youtube.com/watch?v=N9uX1Rsuo7g&t=0)

### הרעיון המרכזי
כיום ידוע שתהליכים מנטליים (זיכרון, חשיבה, רגשות) **נוצרים במוח** - חומר ביולוגי יוצר ישות "לא חומרית".

### רדוקציוניזם
התיאור הביולוגי של תהליכים פסיכולוגיים:
- במקום "אני שמח" → "משתחרר לי סרוטונין במוח"
- במקום "בא לי שוקולד" → "משתחרר לי דופמין במוח"

**חשוב לזכור:** לתיאור הביולוגי אין מעמד *גבוה יותר* מהתיאור הפסיכולוגי - שניהם מתארים את אותו המצב בדרכים שונות!

### הקשר הדו-כיווני
| כיוון | דוגמה |
|-------|-------|
| מוח → פסיכולוגיה | תרופה נגד דיכאון משנה כימיה מוחית → שיפור מצב רוח |
| פסיכולוגיה → מוח | טיפול פסיכולוגי → שינוי בפעילות אזורי מוח |

### מחקר הפוביה מעכבישים
[📺 צפי בסרטון](https://www.youtube.com/watch?v=N9uX1Rsuo7g&t=191)

- נבדקו אנשים עם פוביה מעכבישים
- קבוצה אחת עברה טיפול קוגניטיבי-התנהגותי (CBT)
- **תוצאה:** הטיפול הפסיכולוגי (ללא תרופות!) גרם לירידה בפעילות אזורי הפחד במוח
- **מסקנה:** המוח משתנה כל הזמן בעקבות חוויות, למידה וטיפול

## איך המוח עובד - הנוירון
[📺 צפי בסרטון](https://www.youtube.com/watch?v=4_QB_IfLaCA&t=0)

### המבנה הבסיסי של נוירון

```
    דנדריטים     →     גוף התא     →     אקסון     →     כפתורים טרמינליים
   (קבלת מידע)                      (העברת סיגנל)        (שחרור נוירוטרנסמיטר)
```

| חלק | תפקיד |
|-----|-------|
| **דנדריטים** | ענפים שמקבלים מידע מנוירונים אחרים |
| **גוף התא** | מכיל את הגרעין, מעבד את המידע |
| **אקסון** | צינור ארוך שמעביר סיגנל חשמלי (יכול להיות מכף הרגל לחוט השדרה!) |
| **כפתורים טרמינליים** | קצה האקסון - משחרר נוירוטרנסמיטר |
| **סינפסה** | המרווח בין שני נוירונים |
| **נוירוטרנסמיטר** | חומר כימי שמעביר מידע בין נוירונים |

### כמות הנוירונים במוח
**10¹² (טריליון!)** נוירונים שמעבירים מידע אחד לשני.

| **נוירון מוטורי** | מעביר פקודות מהמוח/חוט השדרה לשרירים | מזיז את היד |

## מה המוח מלמד על זיכרון
[📺 צפי בסרטון](https://www.youtube.com/watch?v=2eJqCSxvykk&t=0)

### המקרה של H.M.
[📺 צפי בסרטון](https://www.youtube.com/watch?v=2eJqCSxvykk&t=31)

- חולה אפילפסיה קשה שבשנות ה-50 הוציאו לו את ה**היפוקמפוס** משני הצדדים
- האפילפסיה נפתרה, אבל **מיום הניתוח לא יכול היה ליצור זיכרונות חדשים**
- נחקר עד מותו ב-2008 ותרם עצום להבנת הזיכרון

### סוגי זיכרון

```
                        זיכרון
                          │
           ┌──────────────┴──────────────┐
           │                             │
    זיכרון לטווח קצר              זיכרון לטווח ארוך
    (דקות ספורות)                       │
                          ┌─────────────┴─────────────┐
                          │                           │
                    דקלרטיבי                    לא-דקלרטיבי
                 (אפשר לדווח)                (לא מודע)
                          │                           │
              ┌───────────┴───────────┐               │
              │                       │               │
         אפיזודי                סמנטי         ┌───────┴───────┐
      (אירועים)              (עובדות)         │               │
                                          פרוצדורלי    התניה + פריימינג
                                         (מיומנויות)
```

## שיטות מחקר: MRI, EEG, TMS

### MRI - הדמיית תהודה מגנטית
[📺 צפי בסרטון](https://www.youtube.com/watch?v=ZnT65pkA1sY&t=0)

- מודד **צריכת חמצן** באזורים פעילים במוח
- מחלק את המוח ל-**ווקסלים** (קוביות תלת-ממדיות)
- **יתרון:** רזולוציה מרחבית מצוינת (איפה)
- **חיסרון:** רזולוציה זמנית נמוכה (לא יודע מתי בדיוק)

**גילוי:** אזורי פנים במוח (ננסי קנווישר, 1997) - מגיבים לפנים יותר מחפצים

### EEG - אלקטרואנצפלוגרפיה
[📺 צפי בסרטון](https://www.youtube.com/watch?v=OBxTVnGdYhQ&t=0)

- מודד **פעילות חשמלית** ישירות מהקרקפת
- **יתרון:** רזולוציה זמנית מעולה (מילישניות!)
- **חיסרון:** לא יודע בדיוק מאיפה במוח מגיע הסיגנל

**גילוי:** תוך 170 מילישניות המוח כבר מבחין בין פנים לחפצים! (פרופ' שלמה בנטין, 1996)

### TMS - גירוי מגנטי
[📺 צפי בסרטון](https://www.youtube.com/watch?v=rZ_iVXKR6Ow&t=0)

- שדה מגנטי מגרה אזור ספציפי במוח
- **יתרון:** מאפשר לקבוע **קשר סיבתי** (האזור X גורם להתנהגות Y)
- **חיסרון:** יכול לבדוק רק אזור אחד בכל פעם

### נקודות חשובות למבחן

1. **הקשר מוח-נפש דו-כיווני:** גם תרופות וגם טיפול פסיכולוגי משנים את המוח

2. **העברת מידע:**
   - בתוך הנוירון = **חשמלי**
   - בין נוירונים = **כימי** (נוירוטרנסמיטר)

3. **ההמיספרות:**
   - כל צד שולט בצד **הנגדי** של הגוף
   - השפה ב**שמאל** (אצל רוב האנשים)

4. **היפוקמפוס:**
   - לא מאחסן זיכרונות
   - אחראי על **גיבוש** (קונסולידציה)
   - H.M. - לא יכול ליצור זיכרונות חדשים אבל זוכר את העבר

5. **זיכרון לא-דקלרטיבי לא תלוי בהיפוקמפוס:**
   - H.M. השתפר במטלות מוטוריות
   - התניה עבדה
   - פריימינג עבד

6. **שיטות מחקר:**
   - MRI = איפה
   - EEG = מתי
   - TMS = קשר סיבתי
   - **כדאי לשלב!**

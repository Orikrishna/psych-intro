## מהי פסיכופתולוגיה?

[📺 צפי בסרטון](https://www.youtube.com/watch?v=Dy78ayjO3tQ)

### הגדרה
**פסיכופתולוגיה** = פתולוגיה (מחלה) של המיינד = הפרעה נפשית/מנטלית

### שני האתגרים המרכזיים

```
┌─────────────────────────────────────────────────────────────────┐
│                 אתגרי הפסיכופתולוגיה                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   1. אבחון (Diagnosis)                                         │
│      האם האדם סובל מהפרעה? מאיזו?                               │
│      → כלי אבחון צריכים להיות מהימנים ותקפים                   │
│                                                                 │
│   2. טיפול (Treatment)                                         │
│      מה הטיפול המתאים?                                         │
│      → הטיפול צריך להיות אפקטיבי (יעיל)                        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### מה מייחד הפרעות נפשיות?
- מקורן בהפרעה **מוחית** (לא באיברים אחרים)
- אבל עדיין אין בדיקות פיזיולוגיות לאבחון

## ספר ה-DSM

### מהו DSM?
**DSM = Diagnostic and Statistical Manual of Mental Disorders**
- מוציא לאור: ארגון הפסיכיאטרים האמריקאי
- מכיל: כ-300 הפרעות נפשיות מוגדרות
- לכל הפרעה: רשימת סימפטומים מדויקת

### הגדרת הפרעה מנטלית לפי DSM

> "סינדרום (אוסף סימפטומים) שמאופיין בהפרעה משמעותית בקוגניציה, ויסות רגשי או התנהגות, הבאה לידי ביטוי בחוסר תפקוד. מלווה בדרך כלל בסבל משמעותי או פגיעה בתפקוד חברתי/תעסוקתי."

### קטגוריות הפרעות (דוגמאות)

| קטגוריה | דוגמאות |
|---------|---------|
| הפרעות נוירו-התפתחותיות | אוטיזם, פיגור שכלי |
| סכיזופרניה | - |
| הפרעות דיכאון | דיכאון מג'ורי |
| הפרעות חרדה | פוביות, חרדה חברתית, פאניקה |
| OCD | הפרעה אובססיבית-קומפולסיבית |
| הפרעות טראומה | PTSD |
| הפרעות אישיות | - |

### דוגמה: הפרעת פאניקה

**קריטריון A - התקפות פאניקה:**
עלייה חדה בפחד שמגיעה לשיא תוך דקות, עם לפחות 4 מהסימפטומים:
- עלייה בקצב לב
- הזעה, רעד
- קוצר נשימה, תחושת חנק
- כאב בחזה
- בחילה, סחרחורת
- תחושת חוסר שליטה / "אני משתגע/ת"
- פחד ממוות

**קריטריון B - מה קורה אחרי:**
- חשש מתמשך מהתקף נוסף
- שינוי בהתנהגות (הימנעות)

### למה ה-DSM חשוב?

| סיבה | הסבר |
|------|------|
| טיפול מתאים | בחירת הטיפול הנכון להפרעה |
| תקשורת | שפה משותפת לאנשי מקצוע |
| מחקר | הגדרת אוכלוסיית המחקר |
| שירותי רווחה | אבחון מקובל לקבלת שירותים |

### חסרונות ה-DSM
- **סטיגמה** - אנשים לא רוצים "תווית"
- **קושי בסיווג** - לא תמיד מתאים לקטגוריה אחת
- **השפעות תרבותיות** - דוגמה: הומוסקסואליות הוצאה מה-DSM ב-1973

### שכיחות הפרעות נפשיות
- **50%** מהאנשים סבלו מהפרעה כלשהי במהלך חייהם
- **25%** סבלו מהפרעה בשנה האחרונה

## כלי אבחון

### למה אי אפשר להשתמש בבדיקות מוחיות?

```
                 חפיפה בין קבוצות

    פעילות מוחית
         ↑
    10   │        ●●         ●●
     9   │       ●●●●       ●●●●
     8   │      ●●●●●●     ●●●●●●    ← חפיפה!
     7   │     ●●●●●●●●   ●●●●●●●●
     6   │    ●●●●●●●●●● ●●●●●●●●●●
     5   │         ▼           ▼
         └──────────────────────────→
              בריאים      חולים
```

**הבעיה:** יש חפיפה בין הקבוצות - לא ניתן לדעת מערך בודד אם האדם בריא או חולה.

### כלי האבחון הקיימים

| כלי | יתרונות | חסרונות |
|-----|---------|---------|
| **ריאיון לא מובנה** | גמישות, המטופל מוביל | מהימנות נמוכה |
| **ריאיון מובנה (MINI)** | מהימנות גבוהה, אחידות | טכני, פחות "אנושי" |
| **שאלוני דיווח עצמי (BDI)** | קל להעביר, מספרי | אפשר "לשקר", מצמצם לציון |
| **מבחני השלכה (רורשך)** | מגיע ללא-מודע | תוקף בעייתי |

### ריאיון מובנה - MINI

**Mini International Neuropsychiatric Interview**

**דוגמה לשאלות דיכאון:**
1. "האם היית מדוכא/ת רוב היום, כמעט כל יום, בשבועיים האחרונים?"
2. "האם גילית פחות עניין ברוב הדברים?"

→ אם התשובה "לא" לשתיהן = עוברים להפרעה הבאה

### שאלון BDI (Beck Depression Inventory)

- 21 שאלות
- ציון 0-63
- מפותח ע"י אהרון בק

**חסרון:** המטופל יכול להשפיע על התוצאות (למשל כדי לקבל שירותים או להימנע מאבחון)

### מבחן רורשך

**היתרון:** מגיע לתכנים לא מודעים
**הבעיה:**
- רק עם ציינון אקסנר מובנה יש תוקף סביר
- **ככל שמוסיפים פרשנות - התוקף יורד!**
- הפסיכולוג חושב שהוא מוסיף, בפועל הוא גורע

## טיפול פסיכודינמי

### הבסיס - זיגמונד פרויד

**הרעיון המרכזי:** המקור לפתולוגיה = קונפליקט בין תכנים מודעים ולא מודעים.

**מטרת הטיפול:** להגיע למידע ברמה הלא מודעת.

### מאפייני הטיפול הפסיכואנליטי המסורתי

- מפגשים כמעט יומיים
- נמשך **שנים**
- המטופל שוכב על ספה, המטפל מאחוריו
- קבלה לא שיפוטית

### טכניקות

| טכניקה | הסבר |
|--------|------|
| **אסוציאציות חופשיות** | המטופל אומר כל מה שעולה על דעתו |
| **ניתוח חלומות** | חלומות חושפים תכנים לא מודעים |
| **טרנספרנס** | המטופל משליך על המטפל רגשות מהעבר |
| **קאונטר-טרנספרנס** | תגובת המטפל על הטרנספרנס |

### הטיפול הפסיכודינמי המודרני

- פחות אינטנסיבי (פעם בשבוע)
- עדיין עוסק בלא מודע, העברה, עבר
- **המטופל קובע את האג'נדה**

## טיפול קוגניטיבי-התנהגותי CBT

### שני מקורות

```
┌─────────────────────────────────────────────────────────────────┐
│                          CBT                                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   התנהגותי (ג'וזף וולפה)    +    קוגניטיבי (אהרון בק)          │
│   ↓                              ↓                              │
│   חשיפה להתמודדות               שינוי מחשבות                   │
│   עם הימנעות                    אוטומטיות                      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### החלק ההתנהגותי - חשיפה

**הרעיון:** הימנעות מחזקת את הפחד. חשיפה הדרגתית = הכחדה.

**Systematic Desensitization (וולפה):**
- חשיפה הדרגתית לגירוי המפחיד
- בשיתוף פעולה מלא עם המטופל
- הוא קובע את הקצב והעוצמה

**דוגמה - טיפול בהפרעת פאניקה:**
1. הסבר על המנגנון (Psychoeducation)
2. חשיפה לתחושות פיזיולוגיות (דופק, נשימה)
3. חשיפה למקומות שנמנע מהם
4. לומדים: התחושות לא מסוכנות!

### החלק הקוגניטיבי - אהרון בק

**הרעיון:** הפרעות נובעות מהאופן שבו מפרשים אירועים.

**מחשבות אוטומטיות** = פרשנויות אוטומטיות שגורמות לרגשות שליליים

### הטיות קוגניטיביות (רשימת בק)

| הטיה | דוגמה |
|------|-------|
| **חשיבה בשחור-לבן** | "או שאני מצליח/ה לחלוטין או שאני כישלון" |
| **קריאת מחשבות** | "אני יודע/ת שהוא חושב עליי רע" |
| **הכללת יתר** | "זה תמיד קורה לי" |
| **פילטר שלילי** | מתמקדים רק בדבר השלילי |
| **הגדלה/הקטנה** | מגדילים כישלונות, מקטינים הצלחות |

**הטיפול:** זיהוי מחשבות → זיהוי הטיות → בחינת תוקף → שינוי

### הגל השלישי - מיינדפולנס ו-ACT

**הרעיון:** במקום לשנות מחשבות - **לקבל אותן**

**ניסוי הדוב הלבן:**
> "אל תחשבו על דוב לבן בדקה הקרובה"
> → בלתי אפשרי! ככל שמנסים יותר, חושבים יותר

**המסקנה:** לא ניתן לשלוט במחשבות. עדיף ללמוד לקבל אותן.

### השוואה: פסיכודינמי vs CBT

| מאפיין | פסיכודינמי | CBT |
|--------|------------|-----|
| משך | ארוך (שנים) | קצר (3-6 חודשים) |
| מוקד | עבר, לא מודע | הווה, בעיה ספציפית |
| מי מוביל | המטופל | המטפל |
| שיעורי בית | אין | כן, קריטי |
| יציאה מהקליניקה | לא | כן (חשיפות) |
| מטרות | תובנה | שינוי התנהגותי |

## טיפול מבוסס ראיות

### למה צריך מחקר?

**אפקט הפלצבו:** האמונה שמקבלים טיפול יכולה לרפא בפני עצמה!
→ צריך להוכיח שהטיפול עוזר **מעבר** לפלצבו.

### איך בודקים יעילות?

```
1. פרוטוקול אחיד
   ↓
2. חלוקה רנדומלית לקבוצות
   ↓
3. מדידה לפני ואחרי (שאלונים)
   ↓
4. השוואה בין קבוצות
   ↓
5. מעקב לטווח ארוך
```

### מטה-אנליזה

**מה זה?** ניתוח סטטיסטי של הרבה מחקרים יחד.

**דוגמה:** 23 מחקרים על טיפול פסיכודינמי קצר לדיכאון:
- נמצא שיפור בסימפטומים
- השיפור נשמר גם אחרי שנה
- יעיל באותה מידה כמו CBT

### מה למדנו?

| ממצא | משמעות |
|------|--------|
| 2/3 משתפרים בטיפול פסיכולוגי | טיפול עוזר! |
| CBT יעיל במיוחד לחרדה | עדיפות ספציפית |
| פסיכודינמי גם נבדק ונמצא יעיל | לא רק CBT |
| ההתאמה מטפל-מטופל חשובה | הברית הטיפולית |

## מיינדפולנס

### מהי קשיבות (Mindfulness)?

> "מודעות בהירה ולא שיפוטית למתרחש ברגע הנוכחי"

**מקור:** פילוסופיה בודהיסטית
**התאמה מערבית:** ג'ון קבט-זין (MBSR)

### איך מתרגלים?

| תרגיל | מה עושים |
|-------|----------|
| **מדיטציית ישיבה** | התבוננות בנשימה, חזרה כשנודדים |
| **הליכה מודעת** | תשומת לב לכל שלב בהליכה |
| **אכילה מודעת** | הרחה, מרקם, טעם - לאט |
| **יוגה** | קשר תנועה-נשימה |

### מה המחקר מראה?

**השפעות חיוביות:**
- ירידה בלחץ, חרדה, דיכאון
- שיפור במערכת החיסון
- ירידה בסיכוי לאירועי לב
- שיפור בקשב (פחות אימפולסיביות)
- התמודדות טובה יותר עם כאב

**שינויים במוח:**
- שינוי בכמות החומר האפור
- קישוריות טובה יותר בין אזורים
- פחות פעילות ב-Default Mode Network (פחות הרהורים)

### מיינדפולנס ו-ADHD

**ממצא מעניין:** מאפייני ADHD הם **תמונת ראי** של מתרגלי קשיבות!

| מתרגלי קשיבות | אנשים עם ADHD |
|---------------|---------------|
| קישוריות מוחית חזקה | קישוריות חלשה |
| DMN משתתק בפעילות | DMN לא משתתק |
| יחס בטא/תטא גבוה | יחס נמוך |

**המחקר:** סדנת MBSR לסטודנטים עם ADHD → שיפור בקשב!

## ראיון עם פרופ' עומר

### מה מייחד פסיכולוג קליני?

**התשובה:** הזיקה למחקר!
- מקצוע אקדמי = מבוסס מחקר
- ללא זיקה למחקר = מקצוע אלטרנטיבי

### דוגמה לחשיבות המחקר

**ממצא:** טיפול קבוצתי לנוער עבריין **מזיק** יותר מחוסר טיפול!
**למה?** עבריינים לומדים עבריינות אחד מהשני.
→ פסיכולוג שלא מכיר את המחקר עלול להזיק.

### האם טיפול פסיכולוגי עוזר?

> "שני שליש מהאנשים שעוברים טיפול נמצאים במצב יותר טוב ממה שהיו אילו לא קיבלו טיפול."

### מה חשוב בבחירת מטפל?

1. **התאמה אישית** - סימפטיה הדדית, שפה משותפת
2. **הברית הטיפולית** - הסכמה על יעדים
3. **קשיבות ליעדים שלך** - המטפל שומע מה אתה רוצה

### על תוקף הרורשך

**הממצא האכזרי:**
> "ככל שמשקל האלמנט הפרשני עולה, כך יורד התוקף."

- הציינון הטכני (אקסנר) = תקף
- הפרשנות של הפסיכולוג = מוריד תוקף
- ככל שיותר פרשנות = תוקף יורד יותר

```
┌─────────────────────────────────────────────────────────────────┐
```

### 📝 נקודות חשובות למבחן

1. **DSM מתאר, לא מסביר** - רשימת סימפטומים, לא סיבות
2. **אי אפשר לאבחן עם MRI** - יש חפיפה בין בריאים לחולים
3. **ריאיון מובנה > לא מובנה** - מהימנות גבוהה יותר
4. **רורשך תקף רק עם ציינון** - פרשנות מורידה תוקף
5. **CBT = חשיפה + עבודה קוגניטיבית**
6. **פסיכודינמי = לא מודע + טרנספרנס + עבר**
7. **2/3 משתפרים בטיפול** - טיפול פסיכולוגי עוזר!
8. **התאמה מטפל-מטופל חשובה** - הברית הטיפולית
9. **מיינדפולנס = קבלה** - לא ניסיון לשנות מחשבות
10. **מדע משתנה** - הידע של היום יתעדכן

## השפעת אחרים על ביצועים

[📺 צפי בסרטון](https://www.youtube.com/watch?v=_CYuUhmedqM)

### השאלה המרכזית
האם נוכחותם של אחרים משפרת או פוגעת בביצועים שלנו?

### הניסוי של טריפלט (Triplett, 1898)
הניסוי הראשון בפסיכולוגיה החברתית!

**מה בדק?** האם רוכבי אופניים רוכבים מהר יותר כשהם מתחרים מול אחרים או כשהם רוכבים לבד.

**הממצאים:**
- רוכבים שהתחרו מול אחרים היו **מהירים יותר** מאלו שרכבו לבד
- הניסוי שוכפל במעבדה עם ילדים שגלגלו חכות דיג - אותה תוצאה

### הבעיה: תוצאות סותרות
מחקרים מאוחרים יותר מצאו תוצאות הפוכות - לפעמים נוכחות אחרים **פוגעת** בביצועים!

### הפתרון של זאיונץ (Zajonc, 1969) - ניסוי המקקים

```
┌─────────────────────────────────────────────────────────────────┐
│                    תיאוריית זאיונץ                              │
├─────────────────────────────────────────────────────────────────┤
│   נוכחות אחרים → עוררות פיזיולוגית ↑ → תגובה דומיננטית ↑        │
├─────────────────────────────────────────────────────────────────┤
│   משימה קלה/מוכרת → התגובה הנכונה היא הדומיננטית → ביצוע ↑     │
│   משימה קשה/חדשה → התגובה השגויה היא הדומיננטית → ביצוע ↓      │
└─────────────────────────────────────────────────────────────────┘
```

**ניסוי המקקים:**
| תנאי | מסלול פשוט | מסלול מורכב |
|------|------------|-------------|
| מקק לבד | איטי | מהיר |
| מקק עם "קהל" מקקים | **מהיר יותר** | **איטי יותר** |

**המסקנה:** גם אצל מקקים! נוכחות אחרים מגבירה עוררות, שמשפרת ביצוע במשימות קלות ופוגעת במשימות קשות.

### מושגים חשובים

| מושג | הסבר |
|------|------|
| **הקלה חברתית (Social Facilitation)** | שיפור בביצועים בנוכחות אחרים - קורה במשימות קלות/מוכרות |
| **עיכוב חברתי (Social Inhibition)** | פגיעה בביצועים בנוכחות אחרים - קורה במשימות קשות/חדשות |
| **תגובה דומיננטית** | התגובה הראשונה והאוטומטית שעולה לנו |
| **עוררות פיזיולוגית** | מצב של ערנות מוגברת בגוף |

## השפעת אחרים על שיפוטים - קונפורמיות

[📺 צפי בסרטון](https://www.youtube.com/watch?v=vNjPEFO165c)

### מהי קונפורמיות?
**קונפורמיות (Conformity)** = שינוי בעמדות או בהתנהגות כתוצאה מלחץ חברתי אמיתי או מדומיין.

### ניסוי שריף - האפקט האוטוקינטי (1936)

**התופעה:** כשמביטים בנקודת אור קטנה בחדר חשוך, היא נראית זזה (אשליה אופטית).

**הניסוי:**
1. שלב ראשון - נבדקים לבד: כל אחד העריך טווח תזוזה שונה (2-6 אינצ')
2. שלב שני - נבדקים בקבוצה: ההערכות **התכנסו** לנורמה קבוצתית משותפת!

**המסקנה:** כשאין לנו מידע ברור, אנחנו מסתמכים על אחרים כדי לפרש את המציאות.

### ניסוי אש - הקווים (Asch, 1951)

[📺 צפי בסרטון](https://www.youtube.com/watch?v=vNjPEFO165c&t=198)

**הניסוי המפורסם:**
```
קו מטרה:    |
            |
            |

קווי השוואה:
   A         B         C
   |         |         |
   |         |         |
             |         |
             |
```

- הנבדק יושב עם 7 אנשים נוספים (שחקנים!)
- המשימה: לומר איזה קו שווה לקו המטרה
- השחקנים עונים ראשונים ונותנים תשובה **שגויה בעליל**
- התשובה הנכונה ברורה לחלוטין

**הממצאים המפתיעים:**
| מדד | תוצאה |
|-----|-------|
| אחוז הנבדקים שהתאימו לקבוצה לפחות פעם אחת | **75%** |
| אחוז התשובות השגויות הכללי | **37%** |
| אחוז שגיאות כשנבדקים לבד | **1%** בלבד |

### שני סוגי השפעה חברתית

| סוג | הסבר | דוגמה |
|-----|------|-------|
| **השפעה אינפורמטיבית** | אנחנו מקבלים את דעת הרוב כי אנחנו חושבים שהם **יודעים משהו** שאנחנו לא | ניסוי שריף - מצב עמום |
| **השפעה נורמטיבית** | אנחנו מתאימים את עצמנו כי אנחנו רוצים **להתקבל** ולא להיראות שונים | ניסוי אש - התשובה ברורה |

### מתי קונפורמיות גוברת?

- גודל הקבוצה גדל (עד 3-4 אנשים, אח"כ מתייצב)
- הקבוצה פה-אחד (אפילו מתנגד אחד מפחית קונפורמיות ב-80%!)
- המשימה קשה או עמומה
- התרבות קולקטיביסטית (לעומת אינדיבידואליסטית)

## השפעת אחרים על התנהגות - עזרה

[📺 צפי בסרטון](https://www.youtube.com/watch?v=xBwDXxMlfN8)

### המקרה של קיטי ג'נובז (1964)

**מה קרה:** אישה צעירה נרצחה בניו יורק. לפי הדיווחים הראשוניים, 38 שכנים שמעו את צעקותיה במשך 35 דקות - ואף אחד לא התקשר למשטרה.

**הערה:** מחקר מאוחר יותר הראה שהסיפור הוגזם בתקשורת, אבל הוא עורר את המחקר על **אפקט הצופה**.

### אפקט הצופה (Bystander Effect)

**ההגדרה:** ככל שיש יותר אנשים שעדים לאירוע חירום, כך הסיכוי שמישהו יעזור **יורד**.

### ניסויי לטאנה ודארלי

**ניסוי 1 - עשן בחדר:**
- נבדקים ממלאים שאלון בחדר
- פתאום עשן מתחיל להיכנס מהפתח
- האם יוצאים להזהיר?

| תנאי | אחוז שדיווחו |
|------|-------------|
| נבדק לבד | **75%** |
| נבדק + 2 שחקנים אדישים | **10%** בלבד! |

**ניסוי 2 - התקף אפילפסיה:**
- נבדקים משוחחים באינטרקום עם "נבדקים אחרים"
- אחד מהם מדמה התקף אפילפסיה
- האם הולכים לעזור?

| מספר אנשים באינטרקום | אחוז שעזרו | זמן תגובה ממוצע |
|---------------------|-----------|----------------|
| 2 (נבדק + קורבן) | **85%** | 52 שניות |
| 3 | 62% | - |
| 6 | **31%** בלבד | 166 שניות |

### למה זה קורה? שני מנגנונים

```
┌─────────────────────────────────────────────────────────────────┐
│              למה אנשים לא עוזרים בקבוצה?                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  1. פלורליסטי אי-ידיעה (Pluralistic Ignorance)                 │
│     "אם אף אחד לא מגיב, כנראה שזה לא מצב חירום"                 │
│     → אנחנו מסתכלים על אחרים כדי לפרש את המצב                  │
│                                                                 │
│  2. פיזור אחריות (Diffusion of Responsibility)                 │
│     "מישהו אחר בטח יטפל בזה"                                   │
│     → האחריות מתחלקת בין כולם                                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### מה כן עוזר?
- להיות לבד עם הקורבן
- המצב ברור שזה חירום
- יש לנו את הכישורים לעזור
- אנחנו מרגישים אחריות אישית

## השפעת אחרים על התנהגות - ציות

[📺 צפי בסרטון](https://www.youtube.com/watch?v=eeSlILpkvLU)

### ניסוי מילגרם (1963) - הניסוי המפורסם ביותר בפסיכולוגיה

**הרקע:** מילגרם רצה להבין איך אנשים רגילים ביצעו זוועות בשואה. האם הם היו "רעים" או שציות לסמכות הוא תכונה אנושית כללית?

**מה הנבדקים חשבו:**
- הם משתתפים בניסוי על למידה
- הם "המורים" שצריכים לתת שוקים חשמליים ל"תלמיד" כשהוא טועה
- השוקים עולים מ-15 וולט עד **450 וולט** (מסומן "XXX - מסוכן")

**מה באמת קרה:**
- ה"תלמיד" היה שחקן ולא קיבל שוקים אמיתיים
- הנסיין בחלוק לבן הורה להמשיך גם כש"התלמיד" צועק מכאב

### התוצאות המזעזעות

| רמת מתח | אחוז שהמשיכו עד לשם |
|---------|---------------------|
| 300V - "התלמיד" מפסיק לענות | 80% |
| 375V - שקט מוחלט | 65% |
| **450V - המקסימום** | **65%** המשיכו עד הסוף! |

**לפני הניסוי:** מילגרם שאל פסיכיאטרים מה יקרה - הם חזו ש-1% בלבד יגיעו למקסימום.

### מה גרם לציות?

| גורם | הסבר |
|------|------|
| **סמכות לגיטימית** | הנסיין בחלוק לבן, אוניברסיטת ייל |
| **הדרגתיות** | התחילו מ-15V, עלייה קטנה בכל פעם |
| **מרחק מהקורבן** | לא רואים אותו, רק שומעים |
| **קרבה לסמכות** | הנסיין נמצא ליד הנבדק |

### וריאציות של הניסוי

| שינוי בניסוי | אחוז ציות מלא |
|-------------|---------------|
| ניסוי מקורי | 65% |
| הנסיין נותן הוראות בטלפון | 21% |
| הנבדק צריך להחזיק את יד הקורבן | 30% |
| שני נסיינים חולקים | 0% |
| נבדק אחר מסרב | 10% |

### המסר של מילגרם

> "אנשים רגילים, שרק עושים את העבודה שלהם, ללא כל עוינות מיוחדת, יכולים להפוך לסוכנים בתהליך הרסני נורא."

**זה לא על אנשים רעים - זה על כוחו של המצב.**

## קוגניציה חברתית

[📺 צפי בסרטון](https://www.youtube.com/watch?v=20mlYALOZRY)

### מהי קוגניציה חברתית?
המחקר של איך אנחנו **תופסים, מפרשים וזוכרים** מידע על אנשים אחרים ועל עצמנו.

### רושם ראשוני - ניסוי אש (1946)

**הניסוי:** שתי קבוצות קיבלו רשימת תכונות על אדם:
- קבוצה א': חכם, מיומן, חרוץ, **חם**, נחוש, מעשי, זהיר
- קבוצה ב': חכם, מיומן, חרוץ, **קר**, נחוש, מעשי, זהיר

**התוצאה:** ההבדל במילה אחת (חם/קר) שינה דרמטית את הרושם הכללי!
- "חם" → נתפס כנדיב, הומוריסטי, חברותי
- "קר" → נתפס כקמצן, רציני, מרוחק

**תכונות מרכזיות (Central Traits):** תכונות מסוימות (כמו חם/קר) משפיעות על הפרשנות של כל התכונות האחרות.

### הטיית הייחוס הבסיסית (Fundamental Attribution Error)

**ההגדרה:** הנטייה לייחס את התנהגות אחרים ל**תכונות פנימיות** שלהם (אופי), במקום למצב החיצוני.

**דוגמה:**
```
מישהו מאחר לפגישה:
├── מה אנחנו חושבים: "הוא לא אחראי, לא אכפת לו"
└── מה יכול להיות: פקק תנועה, בעיה אישית, טעות בלו"ז
```

**למה זה קורה?**
- ההתנהגות בולטת לעין
- הנסיבות פחות נראות
- קל יותר קוגניטיבית

### אפקט ההילה (Halo Effect)

**ההגדרה:** הנטייה שרושם חיובי בתחום אחד ישפיע על השיפוט שלנו בתחומים אחרים.

**דוגמאות:**
- אנשים יפים נתפסים כחכמים וטובים יותר
- סלבריטאים נתפסים כמומחים בתחומים שאינם קשורים לתהילתם
- תלמיד שכותב יפה נתפס כחכם יותר

## סטריאוטיפים

[📺 צפי בסרטון](https://www.youtube.com/watch?v=wH39JtWEp_g)

### מהו סטריאוטיפ?
**סטריאוטיפ** = אמונה כללית על קבוצה מסוימת של אנשים, שמיוחסת לכל חברי הקבוצה.

### מודל תוכן הסטריאוטיפים (Stereotype Content Model)

**פרופ' שנאבל מאוניברסיטת תל אביב** מסבירה שסטריאוטיפים מאורגנים על שני צירים:

```
                        חם/חברותי
                            ↑
                            |
        קשישים, נכים    |    המעמד הגבוה שלנו
        (רחמים)          |    (הערצה)
                            |
    קר ←────────────────+────────────────→ מוכשר
                            |
        עניים, מהגרים    |    עשירים, מקצוענים
        (בוז)            |    (קנאה)
                            |
                            ↓
                        קר/לא חברותי
```

| סטריאוטיפ | חם? | מוכשר? | רגש שמעורר |
|-----------|-----|--------|------------|
| קשישים | ✓ | ✗ | רחמים |
| עשירים (לא מהקבוצה שלנו) | ✗ | ✓ | קנאה |
| מהגרים עניים | ✗ | ✗ | בוז |
| "שלנו" (In-group) | ✓ | ✓ | גאווה |

### סטריאוטיפים סמויים - מבחן IAT

**IAT (Implicit Association Test)** = מבחן שמודד סטריאוטיפים לא מודעים.

**איך זה עובד:**
1. מקישים מהר על מילים/תמונות
2. משייכים לקטגוריות (טוב/רע, שחור/לבן)
3. מודדים את **מהירות התגובה**

**הרעיון:** קל יותר לשייך דברים שכבר מקושרים במוח שלנו.
- אם מהר יותר לשייך "לבן+טוב" מאשר "שחור+טוב" → יש הטיה סמויה

**חשוב:** רוב האנשים מראים הטיות ב-IAT, גם אם הם מאמינים בשוויון!

### איום הסטריאוטיפ (Stereotype Threat)

**ההגדרה:** הפחד לאשר סטריאוטיפ שלילי על הקבוצה שלנו, שגורם בפועל לביצוע גרוע יותר.

**ניסוי קלאסי:**
| קבוצה | הוראות | ביצוע במתמטיקה |
|-------|--------|----------------|
| נשים | "מבחן יכולת מתמטית" | נמוך יותר |
| נשים | "מבחן על פתרון בעיות" | **שווה לגברים** |

**המנגנון:**
```
מודעות לסטריאוטיפ → חרדה → משאבים קוגניטיביים לחרדה → פחות משאבים למשימה → ביצוע ירוד
```

**דוגמאות נוספות:**
- שחורים במבחני אינטליגנציה
- קשישים במבחני זיכרון
- סטודנטים ממעמד נמוך במבחנים אקדמיים

## סיכום ומושגי מפתח

[📺 צפי בסרטון](https://www.youtube.com/watch?v=5B0in_FEbK4)

### הנושאים המרכזיים של השיעור

```
┌─────────────────────────────────────────────────────────────────┐
│                  פסיכולוגיה חברתית - סיכום                      │
├─────────────────────────────────────────────────────────────────┤
│  איך אחרים משפיעים עלינו?                                      │
│                                                                 │
│  1. על הביצועים → הקלה/עיכוב חברתי (זאיונץ)                    │
│  2. על השיפוטים → קונפורמיות (שריף, אש)                        │
```
